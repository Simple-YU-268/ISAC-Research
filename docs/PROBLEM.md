# Cell-Free ISAC 完整问题建模

---

## 通信链路问题 (Communication)

### 信号模型

```
y_k = Σ_m h_mk^H w_mk s_k + Σ_j≠k h_mj^H w_mj s_j + n_k
```

其中:
- `y_k` - 用户k接收信号
- `h_mk` - AP m到用户k的信道 (M×K×Nt)
- `w_mk` - AP m到用户k的波束成形向量 (Nt×1)
- `s_k` - 用户k的发送信号 (功率1)
- `n_k ~ CN(0, σ²)` - 噪声

### SINR 约束

```
SINR_k = |Σ_m h_mk^H w_mk|² / (Σ_j≠k |Σ_m h_mk^H w_mj|² + σ²) ≥ γ_comm
```

### 功率约束

```
Σ_k ||w_mk||² ≤ P_m,max  (每AP功率)
Σ_m Σ_k ||w_mk||² ≤ P_total   (总功率)
```

## 感知链路问题 (Sensing)

### 信号模型 (双站雷达)

```
y_p = Σ_m g_mp^H Z_m g_mp + n_p
```

其中:
- `g_mp` - AP m到目标p的信道 (M×P×Nt)
- `Z_m = Σ_t w_t w_t^H` - 感知协方差矩阵

### SNR 约束

```
SNR_p = |Σ_m g_mp^H Z_m g_mp|² / σ² ≥ γ_sens
```

### CRB 约束 (Cramér-Rao Bound)

```
CRB_p = σ² / (|Σ_m Σ_k g_mk^H w_mk|²) ≤ Γ
```

---

## 完整优化问题

### 决策变量

```
{w_mk}  - 通信波束 (M×K×Nt)
{Z_m}   - 感知协方差 (M×Nt×Nt)
```

### 目标函数

```
min  Σ_m Σ_k ||w_mk||² + tr(Z_m)
```

### 约束集

```
1. SINR_k ≥ γ_comm  (通信SINR)
2. SNR_p ≥ γ_sens   (感知SNR)
3. CRB_p ≤ Γ        (CRB上界)
4. Σ_m Σ_k ||w_mk||² + Σ_m tr(Z_m) ≤ P_total  (总功率)
5. ||w_mk||² ≤ P_m,max              (每AP功率)
6. Z_m ≽ 0                       (半正定)
```

---

## 实际约束 (Imperfect CSI)

### 信道估计误差模型

```
ĥ = h + e,  e ~ CN(0, ε²I)
```

约束改为**最坏情况**：

```
SINR_k^min = min_ĥ SINR(ĥ) ≥ γ_comm
```
